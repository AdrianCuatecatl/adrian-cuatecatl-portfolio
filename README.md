### 🔹 Pipeline ETL de Datos Financieros

**Problema**
La información provenía de múltiples fuentes con inconsistencias, dificultando su análisis y explotación.

**Solución**
Participación en el diseño y desarrollo de un pipeline ETL utilizando SQL Server y Python, estructurado en capas (staging → transformación → modelo final).

**Tecnologías**

* SQL Server (Stored Procedures, CTEs, optimización de queries)
* Python (pandas)

**Impacto**

* Generación de datos consistentes y confiables
* Base para dashboards y modelos analíticos

---

### 🔹 Modelo de Scoring Crediticio

**Problema**
La evaluación de riesgo de clientes era limitada y poco estandarizada.

**Solución**
Desarrollo de un modelo de regresión logística para estimar la probabilidad de incumplimiento.

**Tecnologías**

* Python (scikit-learn, pandas, numpy)
* SQL Server

**Resultados**

* AUC ≈ 0.70
* KS ≈ 0.30

**Impacto**

* Mejora en segmentación de riesgo
* Soporte en decisiones de crédito
