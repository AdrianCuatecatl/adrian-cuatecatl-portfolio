### 🔹 Pipeline ETL de Datos Financieros

**Problema**  
La información provenía de múltiples fuentes con inconsistencias, dificultando su análisis y explotación.

**Solución**  
Participé en el diseño y desarrollo de un pipeline ETL utilizando SQL Server y Python, estructurado en capas (staging → transformación → modelo final).

**Tecnologías**  
- SQL Server (Stored Procedures, CTEs, optimización de queries)  
- Python (pandas)  

**Optimización**  
- Uso de índices para mejorar tiempos de consulta  
- Estructuración modular del pipeline  
- Procesamiento eficiente de grandes volúmenes de datos  

**Impacto**  
- Generación de datos consistentes para análisis  
- Base confiable para dashboards y modelos analíticos

### 🔹 Modelo de Scoring Crediticio

**Problema**  
La evaluación de riesgo de clientes era limitada y poco estandarizada.

**Solución**  
Participé en el desarrollo de un modelo de regresión logística para estimar la probabilidad de incumplimiento.

**Tecnologías**  
- Python (scikit-learn, pandas, numpy)  
- SQL Server  

**Resultados**  
- AUC aproximado de ~0.70  
- KS aproximado de ~0.30  

**Impacto**  
- Mejora en la segmentación de riesgo  
- Apoyo en la toma de decisiones de crédito  

---

### 🔹 Funnel de Originación (Power BI)

**Problema**  
No existía visibilidad clara del avance de las solicitudes a lo largo del proceso de originación.

**Solución**  
Desarrollo de un dashboard en Power BI para el seguimiento del funnel, integrando distintas fuentes de datos.

**Tecnologías**  
- Power BI (DAX, modelado de datos)  
- SQL Server  

**Funcionalidades**  
- Seguimiento por etapa del proceso  
- Métricas de conversión  
- Identificación de cuellos de botella  

**Impacto**  
- Mayor visibilidad operativa  
- Mejor toma de decisiones en originación
